# 登录页面

## 背景图片设置

网页背景已设置为使用 `background.jpg` 图片文件。

### 如何添加背景图片：

1. 将上海海港足球俱乐部2025年夺冠照片保存为 `background.jpg`
2. 将图片文件放在项目根目录（与 `index.html` 同一目录）
3. 支持的图片格式：jpg, jpeg, png, webp

### 或者使用网络图片：

如果想使用网络图片，可以修改 `style.css` 文件中的背景图片URL：

```css
background-image: url('https://your-image-url.com/image.jpg');
```

## 功能说明

- 默认用户名和密码：`admin` / `admin`
- 页面加载后自动登录
- 支持手动输入用户名密码登录
